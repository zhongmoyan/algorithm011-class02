题目：

反转一个单链表。

**示例:**

```
输入: 1->2->3->4->5->NULL
输出: 5->4->3->2->1->NULL
```

demo

1.双指针迭代

```java
    public static ListNode reverseList(ListNode head) {
        ListNode pre = null,tmp=null;
        ListNode cur=head;
        while(cur!=null){
            tmp=cur.next; //记录当前节点的下一个节点
            cur.next=pre;//将 cur 的 next 指向 pre
            //pre和cur节点都前进一位
            pre=cur;
            cur=tmp;

        }
        return pre;
    }
```

复杂度 O(n)
